Para executar a aplicação:

No CMD:

virtualenv novo

novo\scripts\activate

pip install Flask

pip install Flask-Login

pip install twilio

pip install matplotlib

pip install --upgrade numpy scipy pillow

py app.py

Acessar pelo browser:

http://localhost:5000/
